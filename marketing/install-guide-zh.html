🔧 手动安装
4 步完成配置
1
安装 CLI
$ npm install -g @geminilight/mindos@latest
2
初始化知识库 交互式引导
$ mindos onboard --install-daemon
# 选择已有知识库路径，自动启动同步服务
3
配置 MCP & Skill
$ mindos mcp install -g -y
$ npx skills add https://github.com/GeminiLight/MindOS \
    --skill mindos-zh -g -y
4
后续更新
$ mindos update
一行命令升级到最新版本
