# MindOS Logo 设计理念与准则

## 1. 核心视觉符号：不对称的无限大 (The Asymmetric Infinity)
MindOS 的 Logo 是对传统无限符号（∞）的现代重构，象征着人类智慧与机器执行力之间的永恒循环与共生。

### 设计要素
*   **左侧：人类端 (Human Side)**
    *   **表现方式：** 3px 细线条 + 虚线 (Dash Array: 2 4)。
    *   **寓意：** 代表人类思维的**非连续性、跳跃性与灵感碎片**。它是探索性的、不完美的，却是一切行动的源头。
*   **右侧：Agent 端 (Agent Side)**
    *   **表现方式：** 4.5px 粗线条 + 实心线条。
    *   **寓意：** 代表 Agent 的**确定性、连续性与强大执行力**。它是逻辑严密的、连贯的，能将人类的碎片化意图放大为实质性成果。
*   **比例逻辑：** 采用非对称比例，右侧 Agent 环（半径约 22px）明显大于左侧人类环（半径约 15px），象征着 **“思维激发，行动放大”** 的杠杆效应。

在左右两环的物理交汇处，嵌套了一个处于最顶层的**四角星芒**。

*   **视觉参数：** 
    *   **尺寸：** 极小尺寸 (2.5px 放射半径)，保持尖锐且精致。
    *   **颜色：** 暖白色 (`#FEF3C7`)。
*   **寓意：** 
    *   **触发瞬间：** 代表 AI 介入并点燃人类灵感的瞬间。
    *   **图层置顶：** 象征着 MindOS 作为“操作系统”对底层知识流的整合与升华。

## 3. 色彩与梯度
*   **品牌主色：** 琥珀金 (`var(--amber)` / `#F59E0B`)。
*   **梯度渐变：** 
    *   **人类侧：** 向左侧逐渐淡出（Opacity 0.8 -> 0.3），模拟思维的模糊性。
    *   **Agent 侧：** 向右侧逐渐增强（Opacity 0.8 -> 1.0），体现工业级的可靠性与交付感。

## 4. 工程与格式准则
*   **横向标准版 (`logo.svg`)：** 80x40 比例，适用于导航栏、侧边栏。
*   **正方形标准版 (`logo-square.svg`)：** 80x80 比例，采用上下补白 (Vertical Centering) 方式实现，不压缩原始图形比例。适用于 Favicon、App Icon。
*   **代码实现：** 始终使用 SVG 格式，确保在缩放时保持线条末端的圆润 (`stroke-linecap="round"`)。
