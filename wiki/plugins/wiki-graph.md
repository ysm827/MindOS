# Wiki Graph

> 将知识库中的 wikilink 关系渲染为力导向图。

## 基本信息

| 字段 | 值 |
|------|---|
| ID | `graph` |
| 图标 | 🕸️ |
| Core | No |
| 入口文件 | 无（全局切换） |
| 匹配规则 | `() => false`（不自动匹配，需手动激活） |

## 激活方式

Wiki Graph 不绑定特定文件，通过 UI 中的全局图谱按钮激活。在任何 Markdown 文件视图中可以切换显示图谱面板。

## 数据来源

- 调用 `/api/graph` 获取全量节点和边数据
- **节点**：知识库中的每个 Markdown 文件
- **边**：文件间的 `[[wikilink]]` 和 Markdown 链接引用

## 交互功能

- 🕸️ **力导向布局**：节点自动布局，拖拽调整位置
- 🔍 **范围过滤**：
  - **Global**：显示全部文件和链接
  - **Local**：只显示当前文件的 2-hop 关系图
- 📍 **当前文件高亮**：正在查看的文件节点突出显示
- 🔗 **点击导航**：点击节点跳转到对应文件
- 🏝️ **孤立节点检测**：识别没有链接关系的文件
- 🗺️ **Mini-map**：大图谱下的缩略导航

## 适用场景

- 知识库结构可视化
- 发现文件间的隐含关系
- 找到孤立的、未被引用的笔记
- 理解知识网络的拓扑结构
