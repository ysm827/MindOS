# AI Briefing

> 基于最近修改的文件，AI 生成每日摘要简报。

## 基本信息

| 字段 | 值 |
|------|---|
| ID | `summary` |
| 图标 | ✨ |
| Core | No |
| 入口文件 | `DAILY.md` |
| 匹配规则 | `/\b(SUMMARY\|summary\|Summary\|BRIEFING\|briefing\|Briefing\|DAILY\|daily\|Daily)\b.*\.md$/i` |

## 文件格式

不依赖文件内容格式。插件激活后会：

1. 调用 API 获取最近修改的文件列表
2. 将文件变更作为上下文发送给 AI
3. AI 生成结构化摘要

文件本身可以为空，也可以包含之前的摘要内容。

```markdown
# Daily Briefing

（打开此文件时会自动生成 AI 摘要）
```

## 交互功能

- ✨ **自动生成**：打开文件时自动触发 AI 摘要
- 🔄 **重新生成**：点击按钮用最新数据重新生成
- 📄 **文件列表**：显示最近修改的文件及时间戳
- 🔗 **文件链接**：点击跳转到对应文件
- 📡 **流式输出**：AI 响应实时显示

## AI 输出内容

- 📋 近期关键变更摘要
- 🔁 重复出现的主题/模式
- 💡 建议的下一步行动
- 📊 活跃度概览

## 适用场景

- 每日知识库回顾
- 团队工作日报自动化
- 识别知识库中的趋势和盲点
