# Backlinks Explorer

> 显示所有引用当前文件的反向链接及上下文。

## 基本信息

| 字段 | 值 |
|------|---|
| ID | `backlinks` |
| 图标 | 🔗 |
| Core | No |
| 入口文件 | `BACKLINKS.md` |
| 匹配规则 | `/\bBACKLINKS\b.*\.md$/i` |

## 文件格式

不依赖文件内容。插件激活后调用 `/api/backlinks?path=<当前文件>` 获取引用数据。

文件可以为空或包含说明文字：

```markdown
# Backlinks

（打开此文件时自动显示所有引用此文件的链接）
```

## 数据来源

- 后端扫描知识库中所有 `.md` 文件
- 检测 `[[wikilink]]` 和标准 Markdown 链接
- 返回引用文件路径、行号、上下文片段

## 交互功能

- 🔗 **反向链接列表**：显示所有引用当前文件的文件
- 📝 **上下文片段**：高亮显示链接出现的具体段落
- 🔍 **Wikilink 高亮**：视觉强调 `[[链接]]` 文本
- 📂 **点击导航**：直接跳转到引用文件

## 适用场景

- 理解一个概念被哪些文件引用
- 知识库的关系梳理
- 重命名/移动文件前评估影响范围
