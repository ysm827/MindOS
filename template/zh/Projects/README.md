## 💡 使用说明

- `Research/`：存放科研项目相关文档
- `Products/`：存放产品项目 ideation 阶段文档，每个产品一个目录

## 📐 更新规则

### 新增

- 新增产品项目时，在 `Products/` 下创建以产品名命名的目录，标准文档包括：`📄 项目计划书.md`、`📌 产品需求文档.md`、`🔬 竞品分析.md`，并在本文件目录结构中同步更新
- 新增科研项目时，在 `Research/` 下创建目录，并在本文件目录结构中同步更新

### 修改 / 删除

- 修改或删除项目目录时，同步更新本文件目录结构
