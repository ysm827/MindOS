## 核心定位速查

| 层级 | 表达 |
|------|------|
| 品类（内部） | 认知外化平台 |
| 品牌（对外） | 认知复利引擎 |
| 一句话 | 和所有 AI 共享你的大脑 |
| 痛点版 | 别再给每个 Agent 重复交代背景了 |
| 差异化 | Cursor 让你写得更快，MindOS 让你想得更深 |
| 共演化 | 不只是 AI 变准——你也在变强 |
