## 核心叙事（一段话版）

你每天和 AI 对话 2-3 小时。在这些对话中你做出了上百个判断——架构决策、代码规范、设计偏好。这是你最高密度的认知资产。但对话关掉那一刻，全部消失了。人和 AI 的关系搞反了——你在伺候 AI，而不是 AI 在积累你。MindOS 把这个关系翻过来。

## 定位速查

| 场景 | 怎么说 |
|------|--------|
| 对用户（获客） | 别再给每个 AI 重复交代背景了 |
| 对用户（留存） | 你的 AI 越用越懂你 |
| 对投资人（开场） | 判断数据是 AI 时代最值钱的资产，没人在收集 |
| 对投资人（飞轮） | 用户来止痛，判断积累，AI 变准，人变清晰——认知复利 |
| 品牌层 | 让你的每次思考都在增值 |
| 差异化 | Cursor 让你写得更快，MindOS 让你想得更深 |
