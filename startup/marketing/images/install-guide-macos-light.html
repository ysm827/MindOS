macOS 安装（3 步搞定）

下载 → 拖拽 → 放行，一分钟搞定
1
下载并打开 DMG
双击 MindOS-mac.dmg 打开安装镜像
2
拖拽安装
把 MindOS.app 拖到 Applications 文件夹
3
首次打开放行
系统偏好设置 → 隐私与安全性 → 点击 "仍要打开"
遇到"已损坏，无法打开"提示？
打开终端执行：
xattr -cr /Applications/MindOS.app
遇到"无法验证开发者"弹窗？
1. 点击 "取消" 关闭弹窗
2. 打开 系统设置 → 隐私与安全性 → 安全性
3. 点击 "仍要打开"
