选择连接模式

⚡

本地模式 / Local Mode

在本机运行 MindOS，需要安装 Node.js

推荐：个人使用或开发测试

🌐

远程模式 / Remote Mode

连接到远程 MindOS 服务器

推荐：团队协作或服务器部署

📦 缺少 MindOS CLI

一键安装 MindOS CLI 即可使用本地模式（需要 Node.js ≥18）
