# 笔记

用于快速记录与轻量笔记。

## 📁 目录结构

```bash
📝 笔记/
├── README.md
├── INSTRUCTION.md
├── 收件箱/
├── 待反馈/
├── 会议/
└── 想法/
```

## 💡 使用说明

- `收件箱/`：临时速记入口，未整理内容先落这里。
- `待反馈/`：需要他人反馈或外部依赖的事项。
- `会议/`：会议纪要、决策点与行动项。
- `想法/`：灵感池，沉淀产品点子、选题与实验想法。
- 稳定知识应定期归档到 `👤 画像/`、`🔄 流程/`、`🚀 项目/` 或 `📚 资源/`。
- 执行与同步规则以 `INSTRUCTION.md` 为准。
