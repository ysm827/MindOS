# My Mind

个人知识操作系统入口。用于索引核心目录，让 Agent 先理解结构再执行。

## 📁 目录结构

```bash
my-mind/
├── INSTRUCTION.md
├── README.md
├── CONFIG.json
├── TODO.md
├── CHANGELOG.md
├── 👤 画像/
├── 📝 笔记/
├── 🔗 关系/
├── 🔄 流程/
├── 📚 资源/
└── 🚀 项目/
```

## 💡 使用说明

- 启动时优先读取：`INSTRUCTION.md` -> `README.md` -> `CONFIG.json`
- 新内容先记在 `📝 笔记/`，稳定后归档到对应目录
- 需要持续更新的目录：`👤 画像/`、`🔗 关系/`、`🚀 项目/`
